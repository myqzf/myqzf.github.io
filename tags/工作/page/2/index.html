关于移动端的流式分页

关于移动端app分页的记录

一般项目中web管理端分页通常使用 页码pageNo和分页大小pageSize进行分页 。

如果app分页采用web端分页的话，通常会出现以下问题。

1、数据重复。

2、数据丢失。

3、效率底。

app应用面向人群为c端用户，数据量一大，limit性能就急剧下降。

所以web端分页方式不太适合。
