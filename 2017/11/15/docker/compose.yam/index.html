Compose.yam文件

Compose.yam文件是定义services、networks、volumes，默认文件与路径./docker-compose.yml.
服务定义包含将被应用到为该服务启动的每个容器的配置，就像 docker run传递命令行参数一样。
网络和卷的定义也类似于docker network create和docker volume create。
