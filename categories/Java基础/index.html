StingJoiner可变字符串拼接

StringJoiner是Java 8中提供的可变字符串类，可以用于字符串拼接。是java.util包中的一个类。

StringJoiner是通过StringBuilder实现的，所以它的性能和StringBuilder差不多，它也是非线程安全的。

如果日常开发中中，需要进行字符串拼接，如何选择？

1、如果只是简单的字符串拼接，考虑直接使用”+”即可。

2、如果是在for循环中进行字符串拼接，考虑使用StringBuilder和StringBuffer。

3、如果是通过一个集合（如List）进行字符串拼接，则考虑使用StringJoiner。

4、如果是对一组数据进行拼接，则可以考虑将其转换成Stream，并使用StringJoiner处理。

Java反编译

Java反编译工具

JD-GUI

JD-GUI 是一个用 C++ 开发的 Java 反编译工具，由 Pavel Kouznetsov开发，支持Windows、Linux和苹果Mac Os三个平台。而且提供了Eclipse平台下的插件JD-Eclipse。JD-GUI不需要安装，直接点击运行，可以反编译jar,class文件。

目前版本是1.4.0，停留在2015年 。可跟进后续更新。

ecd-plugin (在用)

增强型类型反编译器将JD，Jad，FernFlower，CFR，Procyon与Eclipse无缝集成，并允许Java开发人员直接调试没有源代码的类文件。它还集成了eclipse类编辑器，m2e插件，支持Javadoc， 引用搜索，库源附加，字节码视图和JDK8 lambda表达式的语法。

一、eclipse中安装

  1. 启动Eclipse。

  2. 点击 *”Help > Install New Software…”*。

  3. 点击按钮*”Add…”* 添加一个新的存储库。

  4. 输入名称为“Enhanced Class Decompiler Update Site”并输入位置为“ https://ecd-plugin.github.io/update ”，然后点击“OK”按钮。

  5. 选中“Enhanced Class Decompiler” 。

  6. Next, next, next… 最后重启eclipse。

二、配置文件关联

  1. 点击*”Window > Preferences > General > Editors > File Associations”* 。

  2. “\ .class”：默认情况下选择“类反编译器查看器”*。

  3. “\ .class without source”：默认情况下选择“类反编译器查看器”*。

三、配置ecd插件
点击 “Window > Preferences > Java > Decompiler”进行相关配置。

四、卸载ecd插件

  1. 点击*”Help > About Eclipse > Installation Details > Installation Software”*。
  2. 选择 *”Enhanced Class Decompiler”*。
  3. 点击 *”Uninstall…”*。

HashMap与Hashtable

Hashtable与HashMap的几个不同点：

不同点 HashMap Hashtable
键与值 允许一个key为null和任意个value为null 不允许key和value为null
数据结构 数组+链表+红黑树 数组+链表
继承的类 AbstractMap<K,V> Dictionary<K,V>
是否线程安全 否 是
性能 高 底
默认初始化容量 16 11
扩容方式 原始容量x2 原始容量x2 + 1
底层数组的容量为2的整数次幂 要求一定为2的整数次幂 不要求
确认key在数组中的索引的方法 i = (n - 1) & hash; index = (hash & 0x7FFFFFFF) % tab.length;
遍历方式 Iterator(迭代器) Iterator(迭代器)和Enumeration(枚举)
Iterator遍历数组顺序 索引从小到大 索引从大到小

Hashtable已不建议在新代码中使用，不需要线程安全的场合可以用HashMap替换，需要线程安全的场合可以用ConcurrentHashMap替换。
