CodePush部署

简介

CodePush是一个微软开发的云服务器。通过它，开发者可以直接在用户的设备上部署手机应用更新。CodePush相当于一个中心仓库，开发者可以推送当前的更新（包括JS/HTML/CSS/IMAGE等）到CoduPush，然后应用将会查询是否有更新。

安装nodejs

采用源码包编译的方式

1、下载nodejs源码包
https://github.com/nodejs/LTS/

系统环境需求：
gcc and g++ 4.9.4 or newer
Python 2.6 or 2.7
GNU Make 3.81 or newer
查看系统环境是否满足

不满足的话进行安装或者更新。

1、安装nodejs

2、安装npm

3、设置淘宝镜像

采用官方编译好的安装包

1、下载编译好的安装包

2、添加到环境变量
vim /etc/profile

source /etc/profile
3、验证版本

安装 code-push-server

1、全局安装

2、修改配置文件
vim /dbdata/nodejs/node-v8.11.1/lib/node_modules/code-push-server/config/config.js

3、 初始化数据库
会在数据库中创建一个 database 名字是 codepush

4、启动服务

启动完成后，打开 yourIp:3000 进行登录，默认密码为 admin,123456

1、全局安装

2、相关命令

  • 登录： code-push login http://127.0.0.1:3000

  • 修改密码：

  • 注销： code-push logout

  • 列出登陆的token： code-push access-key ls

  • 删除某个 key值： code-push access-key rm

  • 在账号里面添加一个新的app：code-push app add

  • code-push app

可装可不装，如果不用，请把 code-push-server 配置文件中的 codePushWebUrl 这项注释。
