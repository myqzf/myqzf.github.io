MongoDb自动备份

1、新建mongoDB备份存放目录：

2、创建备份脚本
说明：删除命令一定要注意
vim mongodb_ali_auto_backup.sh

3、添加定时任务
vim /etc/crontab

未解决问题：

输出日志重定向。
单独执行脚本时，日志可以重定向到目标文件。
由crontab 定时执行时， 日志文件并没有重定向到目标文件。

18.3.20 已解决 重定向符号由 &> 换成了 >> “/var/log/mongodb/backlog_ali/$(date +”%Y-%m-%d”).log” 2>&1
